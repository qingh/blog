一、mysql启动关闭方式

net start mysql  //启用mysql服务

net stop mysql  //停止mysql服务

mysql -uroot -p //输入密码进入命令行界面

二、修改mysql root 用户密码

mysql>use mysql;

mysql>update user set password=password('123456') where user='root';

mysql>update user set authentication_string=password('123456') where user='root';

mysql>flush privileges;

三、修改mysql编码

mysql>\s  //查看mysql编码

编辑mysql.ini文件

[mysql]

default-character-set=utf8



[mysqld]

character-set-server=utf8

四、退出mysql命令行的三种方式

1.mysql>exit

2.mysql>quit

3.mysql>\q

五、个性化设置MySQL prompt 提示符

mysql -uroot -p --prompt=qingh

mysql> --->  qingh

prompt mysql>  //变回默认



prompt \u   //root

prompt \h   //localhost

prompt \D   //当前时间

prompt \d   //当前打开的数据库

六、mysql常用语句

mysql>select version(); //当前数据库版本

mysql>select now();     //显示当前时间

mysql>select user();    //显示当前用户

mysql>show engines;  //查看mysql的存储引擎



mysql>show warnings;  //显示上一步操作产生的警告



\c 取消当前语句执行

eg:mysql>select version()\c

注意：上面这条语句不能加分号

七、mysql 日志

mysql> \T C:\Users\Administrator\Desktop\notes\mysql\log.txt    //注意：以上语句，结尾不需要分号



mysql>\t  //结束记录日志。注意：以上语句，结尾不需要分号
